# Data-Analysis-with-Python-MOOC-2022

Course page: https://courses.mooc.fi/org/uh-cs/courses/dap-22

### My solutions for all exercises for all weeks.

## Course format:

Week 1: Python revision

Week 2: More Python revision, NumPy intro

Week 3: More NumPy, Matplotlib intro, Pandas intro

Week 4: More Pandas

Week 5: Even more Pandas, ML: Linear regression

Week 6: ML: Naive Bayes Classification, Clustering, Principal component analysis

Week 7: Project work you choose from the three given choices:
- Biological sequence analysis (part07)
- Regression analysis on medical data (part08)
- Fossil data analysis
